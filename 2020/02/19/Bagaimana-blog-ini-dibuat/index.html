Bagaimana blog ini dibuat?

Author: Yoni Saka Date: February 19, 2020  22:40:06

Blog ini dibuat dengan menggunakan Static Site Generator Hexo. Apa itu Static Site Generator ? bisa dibaca disini. Hexo merupakan static site generator yang berbasis NodeJS yang mendukung beberapa jenis template antara lain EJS, Pug, Handlebars, dll.

Apa saja yang digunakan?

Hampir semua yang digunakan gratis, kecuali domain hehe. Saya menempatkan source code saya di Repositori github saya dan hosting menggunakan Netlify yang saling terintegrasi. Kemudian saya menggunakan tema yang tersedia di Hexo yaitu Chic.

Jadi itu semua yang saya gunakan di web ini. Dahlah :D
