Apa itu semantic UI?

Semantic UI merupakan salah satu framework HTML, CSS, dan Javascript. Di kalangan developer saat ini framework froent-end yang banyak digunakan adalah Bootstrap. Akan tetapi Semantic ini tidak kalah bagus dengan bootstrap, untuk yang bosen dengan tampilan bootstrap dan menyukai tampilan yang elegan mungkin framework ini cocok untuk kamu.
